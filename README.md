## Como rodar a aplicação :computer:
No terminal, clone o projeto:
```
git clone https://github.com/Guihnovo/Move.git
```
Entre na pasta do arquivo
```
cd Move/src
```
Instale as dependências
```
npm install
```
Execute a aplicação
```
npm run start
```
Abrir o navegador no localhost:3000
